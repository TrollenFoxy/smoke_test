Тестирование программы в формате "Smoke test". Тестирование проводилось на платформах web, android, windows. Так же для Smoke test'a была разработана таблица. Success rate считает % успешного выполнения, меняет цвет в зависимости от этого процента. Столбцы Result - 1... Result - 6 выбираются из разворачивающегося списка и окрашиваются в определённый цвет в зависимости от результата пройденного теста.
